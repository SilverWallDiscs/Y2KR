### Requisitos Previos

- [Lenguaje/framework, Node.js].

### Pasos para Instalar

1. **Clona el repositorio**:
   ```bash
   git clone https://github.com/SilverWallDiscs/Y2KR.git
   cd Y2KR
   ```

2. **Instala las dependencias**:
   - Para Node.js:
     ```bash
     npm install
     ```
 

3. **Ejecuta el proyecto**:
   ```bash
   npm run dev 

   ```
